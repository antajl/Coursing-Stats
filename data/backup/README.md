# Бэкапы данных Coursing Stats

Папка для **ручных снимков** перед рискованными операциями с D1 (удаление дубликатов, миграции, массовый reparse).

Каждая подпапка — отдельная операция с собственным `README.md`: что внутри, зачем создано, как откатить.

## Именование

`YYYY-MM-DD-краткое-описание/`

Пример: `2026-07-03-calendar-dedupe/`

## Не коммитить в git

Содержимое `data/backup/` (кроме этого README) — локальные/операционные дампы. Добавляйте в `.gitignore` при необходимости.
