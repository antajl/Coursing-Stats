## Анализ проблемы

### 1. Проблема с fetchWin1251

**Причина:** Встроенный `fetch` в Node.js возвращает Uint8Array, который нельзя передать напрямую в `Buffer.from()`.

**Решение:** Использовать `iconv Buffer.from()` или перейти на node-fetch или axios.

```javascript
// Текущий код (сломан):
const buf = Buffer.from(await res.arrayBuffer());

// Исправленный вариант:
import iconv from 'iconv-lite';
const ab = await res.arrayBuffer();
const buf = iconv.Buffer.from(ab);
```

## Риски

1. **Кодировка**: если не распознать windows-1251, кириллица превратится в мусор
2. **Структура страниц**: может меняться между годами
3. **Названия клички**: могут отличаться между событиями (опечатки)
4. **Дубликаты**: одна и та же собака может быть записана по-разному
