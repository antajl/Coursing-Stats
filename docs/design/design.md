## Краткий обзор

Проект имеет сильную дизайн-систему с унифицированной палитрой (`gold`/`steel` алиасы к `camel`/`warm-blue`). Основные страницы (DogProfile, EventResults, SpeedRecords) используют консистентные стили. "Музейный" стиль с тёплыми нейтральными тонами и мягкими границами.

**Общая зрелость продукта:** 7.5/10

## Dark Mode (предложение)

### Принципы адаптации

1. **Фоны:** Инвертировать яркость, сохраняя насыщенность
2. **Текст:** Инвертировать контрастность
3. **Акценты:** Сохранить насыщенность, возможно немного увеличить яркость
4. **Границы:** Использовать более светлые тона для видимости на тёмном фоне

## Элементы, которые могут отличаться между light и dark mode

### 1. Фоны
- **Light:** `bg-cream-50` для основного фона
- **Dark:** `bg-cream-900` или более тёмный вариант

### 2. Карточки
- **Light:** `bg-white` с `border-old-money-200`
- **Dark:** `bg-old-money-500` с `border-old-money-600`

### 3. Градиенты
- **Light:** `bg-gradient-to-br from-old-money-50 to-old-money-100`
- **Dark:** `bg-gradient-to-br from-old-money-600 to-old-money-700`

### 4. Текст
- **Light:** `text-charcoal-900` для основного текста
- **Dark:** `text-charcoal-50` или `text-charcoal-100`

### 5. Акценты
- **Light:** `text-camel-700` для primary
- **Dark:** `text-camel-400` или `text-camel-300` (более яркий)

### 6. Hover состояния
- **Light:** `hover:bg-cream-50`
- **Dark:** `hover:bg-old-money-400` или `hover:bg-cream-800`

### 7. Focus ring
- **Light:** `ring-camel-100`
- **Dark:** `ring-camel-900` или `ring-camel-950`

### 8. Skeleton loaders
- **Light:** `bg-cream-200` с `animate-pulse`
- **Dark:** `bg-old-money-400` с `animate-pulse`

### 9. Empty states
- **Light:** `text-gray-600` для описания
- **Dark:** `text-old-money-300` для описания

### 10. Error states
- **Light:** `text-red-500`, `bg-red-50`
- **Dark:** `text-red-400`, `bg-red-900`

## Оценка по направлениям

### Визуальный стиль: 8/10

Сильная палитра, хороший "музейный" стиль на основных страницах. Data-heavy страницы выбиваются кастомными тенями.

### Консистентность: 6/10

Основные страницы консистентны, но data-heavy страницы (Events, Judges, TopDogs) используют разные тени и границы.

### UX и читаемость: 8/10

Хорошая структура, читаемые шрифты, хорошая адаптивность. Интуитивная навигация.

### Адаптивность: 8/10

Полная адаптация для мобильных, таблицы заменяются на карточки. FiltersDropdown улучшен для маленьких экранов.

### Цветовая палитра: 9/10

Отлично подобранная палитра с `gold`/`steel` алиасами. Консистентное использование акцентных цветов (`camel` для primary, `warm-blue` для secondary).

### Типографика: 8/10

Единый sans-serif шрифт (Montserrat) с `tracking-tight` для заголовков. Хорошая иерархия размеров.

### Интерактивность: 8/10

Хорошие hover-состояния, плавные переходы. Tooltip с умным позиционированием.

### Доступность (Accessibility): 7/10

- ✅ Семантические теги
- ✅ Клавиатурная навигация (native select)
- ⚠️ Отсутствие ARIA-меток в некоторых интерактивных элементах
- ⚠️ Цветовой контраст можно улучшить для некоторых текстовых элементов

### Производительность (Performance): 7/10

- ✅ React Query для кэширования данных
- ⚠️ Большой бандл (727 kB после минификации)
- ⚠️ Отсутствие code-splitting для маршрутов

## Реализация в Tailwind

### Конфигурация dark mode

```javascript
// tailwind.config.js
module.exports = {
  darkMode: 'class', // или 'media' для автоматического следования за системой
  // ...
}
```

### Пример использования

```jsx
<div className="bg-cream-50 dark:bg-cream-900">
  <h1 className="text-charcoal-900 dark:text-charcoal-50">Заголовок</h1>
  <button className="bg-camel-600 text-white dark:bg-camel-500">
    Кнопка
  </button>
</div>
```
