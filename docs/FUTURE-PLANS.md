### Ссылка на родословную (breedarchive.com)
**Описание:** Добавить возможность перейти на профиль собаки на breedarchive.com из рейтинга.

**Анализ:**
- Прямая ссылка на профиль автоматически **невозможна** — URL содержит случайный UUID из их базы, который нельзя вычислить из клички
- Пример: `https://saluki.breedarchive.com/animal/view/emul-de-gepard-gelila-al-rawda-699f7593-397c-4b71-89a7-09e352f30907`

**Рабочий компромисс:** Ссылка на поиск с предзаполненной латинской частью клички — человеку нужно лишь кликнуть на нужный результат поиска.

**Что нужно проверить перед реализацией:**
1. Точный параметр URL в форме поиска на `saluki.breedarchive.com/animal/index` (поле "Registered name") — нужно открыть страницу и посмотреть имя параметра в коде формы
2. Поддомены под каждую породу свои (`saluki.breedarchive.com`, `sloughi.breedarchive.com` — подтверждены). Нужно составить маппинг `порода → поддомен` и обрабатывать случай когда поддомена для породы нет (не показывать кнопку или показывать общую ссылку)

**Файл для реализации:** `frontend/src/pages/DogProfile.tsx`, возможно компонент `DogName`

**Приоритет:** Низкий
