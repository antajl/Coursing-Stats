## Проблемы с путями

⚠️ **Важно:** В `parse-results-coursing.mjs` импорт:
```javascript
import { fetchWin1251 } from "../lib/fetch-win1251.mjs";
```

Но файл `fetch-win1251.mjs` находится в корне, а не в `lib/`. Нужно либо:
1. Переместить `fetch-win1251.mjs` в `lib/`
2. Исправить импорт на `./fetch-win1251.mjs`

## Большие файлы (>2000 строк)

Текущие файлы не превышают 2000 строк, но при росте проекта нужно следить за:
- `schema.sql` — может вырасти при добавлении view/индексов
- Парсеры для разных типов событий — могут стать большими

Рекомендации по разбивке см. в `06-ai-guidelines.md`.
