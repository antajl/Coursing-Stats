## 2026-07-03 (ночь) — Статистика Донино: 350 м из coursing_records

- Статистика **350 м** (время, средняя/лучшая скорость на трассе) — только из `coursing_records` ([Google Sheet `1hpdA8vl…`](https://docs.google.com/spreadsheets/d/1hpdA8vlIfeECgpnPvuk5xfezPsdUh1EXULjeATAF9dw))
- **Документация:** в `SPEED_RECORDS.md` — раздел «Два вида замеров»: замер скорости (ошейник, курсинг) vs бега 350 м (секундомер); одна собака может быть в обоих листах, метрики не смешиваются

**Проблема:** вкладки «Порода / Пол / Год» в `Stats.tsx` считали «Время за 350м» как `1260 / км/ч` из таблицы **замеров скорости** — получались заниженные секунды (напр. 19,69 с у уиппетов вместо ~22,8 с).

**Решение:**
- Статистика **350 м** (время, средняя/лучшая скорость на трассе) — только из `coursing_records` ([Google Sheet `1hpdA8vl…`](https://docs.google.com/spreadsheets/d/1hpdA8vlIfeECgpnPvuk5xfezPsdUh1EXULjeATAF9dw))
- Вкладка **Обзор** без изменений — гистограмма и агрегаты по `speed_records` (км/ч)
- `recordDates.ts`: `coursingTimesToStats`, `time350ToSpeedKmh`, тесты в `record-dates.test.ts`
- Документация: `SPEED_RECORDS.md`, `PARSING.md`, `DATABASE.md`, `FRONTEND-MAP.md`, `ARCHITECTURE.md`, `API-REFERENCE.md`
