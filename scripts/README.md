# Скрипты для Windows

Папка содержит вспомогательные скрипты для Windows (.bat файлы).

## Доступные скрипты

### start-servers.bat
Запуск локальных серверов для разработки одной командой.

**Использование:**
```bash
scripts\start-servers.bat
```

**Функции:**
- Запускает Worker бэкенд и фронтенд одновременно через `npm run dev`
- Worker бэкенд на http://127.0.0.1:8787
- Фронтенд на http://localhost:5173

**Примечание:** Для работы требуется установленный Node.js и выполненный `npm install` в корне проекта и в frontend/.

### deploy-to-github.bat
Деплой изменений в GitHub для онлайн-версии сайта.

**Использование:**
```bash
scripts\deploy-to-github.bat
```

**Функции:**
- Добавляет все файлы в git
- Коммитит с сообщением "Auto deploy"
- Пушит в GitHub
- После пуша фронтенд и бэкенд деплоятся автоматически через GitHub Actions
- Сайт будет доступен на https://procoursing.antajl.ru

**Примечание:** Для работы требуется настроенный git remote.
